class morl_problems.Financial(gamma=0.9)[source]¶

This is a multi-objective financial toy problem. The agent can choose to invest into a number of financial products. Each action means to buy or to sell one specific financial product. Therefore, if we have f financial products, there will be 2*f actions. The problem is formulated in a bandit-style setting. This means, we have only one state, in which we remain and can choose to buy or sell assets. The vector-valued reward contains the payouts for the current portfolio with respect to

(payout, risk, flexibility)

class morl_problems.MORLBuridansAss1DProblem(size=3, p=0.9, n_appear=10, gamma=0.9)[source]¶

This problem contains buridans ass domain. An ass starts (usually) in a 3x3 grid in the middle position (1,1) in the top left and the bottom right corner there is a pile of food. if the ass moves away from a visible foodstate, the food in the bigger distance will be stolen with a probability of p. Eeating the food means choosing action “stay” at the field of a food it will be rewarded with following criteria: hunger, lost food, walking distance hunger means

class morl_problems.MORLBuridansAssProblem(size=3, p=0.9, n_appear=10, gamma=1.0)[source]¶

This problem contains buridans ass domain. An ass starts (usually) in a 3x3 grid in the middle position (1,1) in the top left and the bottom right corner there is a pile of food. if the ass moves away from a visible foodstate, the food in the bigger distance will be stolen with a probability of p. Eeating the food means choosing action “stay” at the field of a food it will be rewarded with following criteria: hunger, lost food, walking distance hunger means that after 9 steps without eating a food pile the next action(s) will be rewarded with -1
