class problems.PrisonersDilemma(T=5.0, R=3.0, P=1.0, S=0.0)[source]¶

The classical prisoners dilemma (PD) with a default payout matrix of

prisoner 2

C D
prisoner 1 C (R,R) (S,T)
D (T,S) (P,P)

with T > P > R > S and R > (T + S) / 2.
